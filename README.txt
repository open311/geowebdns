GeoWebDNS
==========


GeoWebDNS is a service inspired by DNS and reverse-geocoding. Given a lat/long,
what is the URI of the standardized API, protocol, or file for a given service
namespace (eg 311 services). 

For more information, see http://wiki.open311.org/GeoWeb_DNS

Status
=======
This is an experimental reference implementation.
